Keeping It Reel

Description

For my ISU for Web Design, I opted to create a website that functions as a miniture movie database. Featuring some of my favourite movies, it provides information for each film, in a minimalistic interface.

Another feature on the site is the directors page. Here, I wrote paragraphs covering some of my favourite directors, functioning as a minor biography. Users are able to read about what makes these directors so great, and what was the cause of their successes.
